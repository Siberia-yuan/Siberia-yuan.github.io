Domain Namespace

Domain Namespace是一个有结构层次的一个树
//Country domain
cs->bupt->edu->cn(country domian) top
…->zw(country domian) top
…->ae(country domian) top
//Generic domain
xx->cs->mit->edu(school domain) top
…->com(company domain) top

//200->166->64->59->in-addr(2nd level domains)->arpa(top level domains)

根的直系子孙被直接称之为top level domain
